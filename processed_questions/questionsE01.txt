კითხვა: რომელ პროგრამირების ენას შეისწევლი?
   a) Java
   b) Python
   c) C++
   d) Ruby
   პასუხი: b

კითხვა: Python-ში რას ასრულებს ოპერატორი `if`?
   a) ოპერაციას სტრიქონებზე
   b) ციკლს
   c) პირობითი გამოსახულებას
   d) მონაცემების შეტანა-გამოტანას
   პასუხი: c

კითხვა: Python-ში როგორ უნდა შემოვიტანოთ (გამოვაცხადოთ) ცვლადი?
   a) variable x
   b) int x
   c) var x
   d) x = 5
   პასუხი: d

კითხვა: Python-ში რომელი სიმბოლო გამოიყენება კომენტირებისათვის?
   a) //
   b) --
   c) #
   d) /*
   პასუხი: c

კითხვა: რა შედეგი ექნება ოპერატორის შესრულებას `print(3 + 5)`?
   a) 35
   b) 8
   c) 15
   d) 3 + 5
   პასუხი: b

კითხვა: Python-ში როგორ გავიგოთ სტრიქონის სიგრძე?
   a) len(string)
   b) string.length()
   c) string.len()
   d) str.length
   პასუხი: a

კითხვა: Python-ში რომელი ოპერატორი გამოიყენება სტრიქონების გასაერთიანებლად (კონკატენაციისათვის)?
   a) +
   b) -
   c) *
   d) /
   პასუხი: a

კითხვა: Python-ის სიაში როგორ გავიგოთ ელემენტების რაოდენობა?

   c) len()
